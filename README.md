# Cryptocurrency Dashboard

### Dataset Used:
- https://coinmarketcap.com/api/ 

## Workflow:
1. API call
2. Store on MongoDB
3. Export data to CSV
4. Flask
5. HTML Script
6. Charts.js used
7. User-Driven interactions added
